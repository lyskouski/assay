## Commands

Start the game:
> start home=Mexico away=Canada

Update score (`id` is taken from `--start`): 
> id=0 home=0 away=0

Finish a game (by `id`):
> finish=0 

Get a summary of all games:
> summary

Get a scoreboard of active games:
> scoreboard

Get a scoreboard (by `id`):
> scoreboard=0
